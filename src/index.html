This project was created to provide a simple way to test your HTML styles.

Its main use case is for visualizing the CSS for personal sites and blogs, but its simplicity keeps it versatile. We recommend styling the following tags when fully-skinning your site:

  • Leaf Tags
    • p
    • h1 , h2 , h3 , h4 , h5 , h6
    • hr
    • table , thead , tr , th , tbody
    • pre > code
  • Container Tags
    • ul , ol , li
    • blockquote
  • Inline Tags
    • strong (bold), em (italic), del (strikethrough)
    • code
    • a
    • img

This list was based on Github Flavored Markdown Spec.
